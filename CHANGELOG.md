# Changelog

All notable changes to Rail Gun will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Message reactions with emoji picker
- Typing indicators
- Message replies/quotes
- Link previews
- Message editing with history
- Message pinning
- Conversation muting
- Disappearing messages (timer settings)
- Read receipts
- @mentions with autocomplete
- Right-click context menu on messages
- Bible reader (NABRE - New Testament)
- Decentralized crypto exchange UI

### Security
- End-to-end encryption via Signal Protocol
- OS keychain integration for token storage
- Local-only key stores

## [0.1.0] - 2024-XX-XX

### Added
- Initial release
- End-to-end encrypted messaging using Signal Protocol
- Communities with text/voice channels
- Direct messages
- User presence (online/away/offline)
- Message status indicators (sending/sent/delivered/read)
- Electron desktop app for macOS
- NestJS API backend
- WebSocket real-time messaging
- User authentication with recovery codes

### Security
- libsignal integration for E2EE
- X3DH key exchange
- Double Ratchet message encryption
- Secure token storage via Electron safeStorage
- Rate limiting on API endpoints
- Session-bound WebSocket authentication
