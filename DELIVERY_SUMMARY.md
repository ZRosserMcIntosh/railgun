## ğŸ” The Three Core Protocols Now Highlighted

### Signal Protocol
**What**: End-to-end encryption framework
**How**: X3DH (initial key exchange) + Double Ratchet (per-message ratcheting)
**Why**: Used by Signal Messenger, WhatsApp, and billions of users
**Result**: Perfect forward secrecy + break-in recovery

### Curve25519
**What**: Elliptic curve cryptography
**How**: ECDH key agreement using Curve25519 curve
**Why**: Modern, fast, side-channel resistant
**Result**: Secure key exchange and digital signatures (Ed25519)

### ChaCha20-Poly1305
**What**: Authenticated encryption (AEAD)
**How**: ChaCha20 stream cipher + Poly1305 authenticator
**Why**: Modern alternative to AES-GCM, RFC standard
**Result**: Confidentiality + authenticity + tamper detection
